# ICTabFragment
Tab menu with page view controller

## Minimum Requirements

iOS9+
Swift 3.0
XCode 8.0

## Installation

### Using CocoaPods

```ruby
source 'https://github.com/ioscodigo/ICTabFragment.git'
use_frameworks!
pod 'ICTabFragment'
```

## Usage

`import ICTabFragment` to use ICTabFragment in your file.
